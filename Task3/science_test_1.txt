Microsoft представила новую версию ОС Windows 10

МОСКВА, 22 января. /ТАСС/. Американская корпорация Microsoft представила новую версию операционной системы Windows 10, которая адаптирована для устройств с разной величиной экрана - от персональных компьютеров до смартфонов.
Операционная система будет бесплатна в течение года для пользователей ОС Windows 7 и Windows 8.1, а также последней версии мобильной платформы Windows Phone.
Как говорится в сообщении корпорации, Windows 10 доступна в режиме для десктопов, когда распознает такие устройств, как мышь и клавиатура, если же они не подключены, то система работает в режиме сенсорного управления. При этом в гибридных устройствах ОС будет сама переключаться в тот или иной режим в зависимости от подключения клавиатуры.

Microsoft представила очки дополненной реальности под названием HoloLens
Также все устройства под управлением Windows 10 могут синхронизироваться в режиме реального времени, приложения будут работать одинаково на каждом из них. Для удобства работы с приложениями предусмотрена функция Snap, которая позволяет на одном экране работать с четырьмя программами.
Одной из особенностей Windows 10 является интеграция технологий голографии для программного обеспечения. С помощью очков дополненной реальности HoloLens, которые были представлены корпорацией, пользователи Windows 10 могут взаимодействовать с голограммами, которые они видят с помощью устройства. Сама Microsoft запустила для Windows 10 приложение HoloStudio, которое позволяет создавать 3D-объекты, а потом распечатывать их с помощью 3D-принтера.
В устройствах на Windows 10 также будет поддержка голосового помощника Cortana. С помощью голосовых команд можно будет получать уведомления, информацию о котировках, новости спорта и др.
Также Microsoft представила новый браузер Spartan, который заменит Explorer в Windows 10. Он поддерживает управление с помощью стилуса, а также возможность оставлять заметки на страницах сайтов и делиться ими с другими пользователями.
В Windows 10 снова появится кнопка "Пуск" со специально отведенным местом для любимых приложений, программ и веб-сайтов. Она может отображаться по всему экрану или работать в компактном режиме, привычном для пользователей Windows 7.
